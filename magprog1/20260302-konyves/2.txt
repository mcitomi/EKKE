Egy könyvnek (Book) van:

-írója (Author, string), min 3 betű, max 20 betű. Vezetéknév és keresztnév. Az első betű nagybetű. Nem lehet üres vagy null.
(string.IsNullOrEmpty) Ha hosszabbat adnak meg, ne dobjon kivételt, hanem a 20 utáni karaktereket vágja le!
-címe (Title, string), min 3 betű, max 50 betű.
- ára, egész szám, maximum 15e Ft, 5-el osztható. Ha beállítják, akkor az isSetPrice mező legye true.
- műfaja (genre), enum: mese, regény, történelem, szakkönyv, külföldi. Meg kell írni. genre_enum a neve!!
- e-book-e.  True vagy false.
- Van-e ára (isSetPrice) bool, private. Legyen egy get-je! (Set nyilván nincs.)
- ID-je is van, az elsőé 1, a többié eggyel nő (mint a tét).
- kiadási időpontja (DateTime vagy DateOnly)- Legfeljebb DateTime.Now.

Az osztálynak legyen 3+1 konstruktora
1. Private!!!! Ezt használjuk a klónozáskor. (Pl. private Book() { }
2. Szerző és cím (itt az isSetPrice értéke legyen false)
3. Szerző, cím és ár
4. Szerző, cím, ár, műfaj, e-book-e, kiadás időpontja. Az e-book false, ha nem adják meg! (bool e-book=false)

Legyen egy Clone property is, lásd lent! Legyen egy ToString() és egy Equals() metódusa is. (Contains miatt)

A genre is egy külön .cs-ben legyen!


A könyveket egy könyvespolcon tároljuk, amelynek:

- van egy konstruktora, paraméter nélküli, de létrehozza az üres könyvek listát!
- van egy feltöltő (Load) metódusa, amely tárolja könyvek egy listáját, melyen nem szerepelhet null érték, 
és egy könyv maximum egyszer lehet rajta,
- van egy fv AddBoak(...), paramétere egy könyv, amelyet a könyvespolc listán kívánunk tárolni.
  (Elvileg itt lehetne többet is, túlterhelés, paraméterektől függően.)
- van egy összár (PriceSum) csak olvasható property, megadja a könyvespolcon tárolt könyvek összértékét,
- van egy fv Search(...) egy paramétere egy string, egy író neve, visszatér könyvek egy listájával, amelyek a könyvespolcon vannak tárolva, és amelyeknek az írója a megadott. ha nem talál ilyen könyvet, akkor 0 elemű listával tér vissza.

public override bool Equals(object obj)
{
    if (obj == this) return true;
    if (obj == null) return false;
    if (obj.GetHashCode() != this.GetHashCode()) return false; 

    Book other = (Book)obj;
    return this.author == other.author &&
           this.title == other.title &&
           this.isEbook == other.isEbook;
}
